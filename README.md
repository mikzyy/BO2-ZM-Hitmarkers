# T6ZM ZM Hitmarkers
a simple gsc to add hitmarkers in zombies.

## compiling yourself
1. download [gsc-tool](https://github.com/xensik/gsc-tool/releases)
2. compile with command `./gsc-tool.exe comp t6 hitmarkers.gsc`
